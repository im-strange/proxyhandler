## Description
Proxyhandler is a Python package used to handle proxies.



### Features 
- Fetch proxies from a file 
- Test proxies (http, https, socket)
- Write result into a new file 

### Installation 
```
pip install git+https://github.com/im-strange/proxyhandler.git
```
